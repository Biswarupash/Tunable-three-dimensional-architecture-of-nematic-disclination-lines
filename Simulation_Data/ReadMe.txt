The simulaion is done using the OpenQmin package. The details of the package can be found here: https://github.com/sussmanLab/open-Qmin.

The directory named "Simulation_Data" contains the files used to generate the figures in the paper. 

Fig1c_d.zip: This zip file contains the data file used to generate Figure 1 C and D.
Fig2A_C.txt: Data file used to generate Figure 2 A and C.
Fig2D_F.zip: This zip file contains the data file used to generate Figure 2 D and F. 
Fig4b.zip  : This zip file contains the data file used to generate Figure 4B.
